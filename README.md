# Projeto Jokenpô - Raciocínio Algorítmico

Este projeto consiste na implementação do famoso jogo "Pedra, Papel ou Tesoura" (Jokenpô) utilizando a linguagem Python. O trabalho foi desenvolvido como parte da disciplina de Raciocínio Algorítmico na PUCPR.

# Funcionalidades

O programa oferece três modalidades de jogo:
1. Humano x Humano: Dois jogadores competem entre si.
2. Humano x Computador: Um jogador contra a inteligência artificial (escolhas randômicas).
3. Computador x Computador: Modo de simulação, ambos os algoritmos competem.

# Regras de Implementação (Restrições)
Para atender aos requisitos acadêmicos da atividade somativa, o código foi desenvolvido seguindo restrições específicas:
* Uso exclusivo de estruturas simples (variáveis, loops, while e condicionais if, else e elif).
* Não foram utilizadas listas, funções customizadas ou dicionários.
* Validação de entrada para garantir que o usuário escolha apenas opções válidas.

# Como rodar o projeto
1. Certifique-se de ter o Python instalado.
2. Clone o repositório ou baixe o arquivo.
3. Execute o comando run.
